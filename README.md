# Proyecto-POWER-BI
Pequeño proyecto de Power BI donde analizamos ventas de electronicos y electrodomesticos, por cada tienda y zona, a traves de un periodo de tiempo

el archivo se divide en 3 visiones, 3 perspectivas, cada una en una hoja. Trata de representar el analisis de ventas, y todo lo relacionado, a diferentes tiendas del pais
